## Prerequisites

Before installing DroidVPS, make sure you have Termux installed on your Android device.

## Step 1: Update Termux

First, update your Termux packages to ensure you have the latest versions:

```bash
pkg update
pkg upgrade
```

## Step 2: Install Dependencies

Install Node.js and Git:

```bash
pkg install nodejs git
```

## Step 3: Clone the Project

Clone the DroidVPS repository:

```bash
git clone https://github.com/hlfr07/DroidVPS.git
cd DroidVPS
```

## Step 4: Setup

Proceed to the [QUICK_START.md](QUICK_START.md) guide for configuration and running instructions.

---

**Note**: Make sure to allow storage permissions in Termux if prompted during installation.
